## Development

Para ejecutar rapidamente el ambiente de desarrollo se pueden ejecutar los siguientes comandos
en la terminal:

```bash
$ git clone git@github.com:ulises-jeremias/proyecto-deep-learning-2019.git
$ cd proyecto-deep-learning-2019
$ yarn
$ yarn build:dev
$ yarn serve:dev
```

o utilizando docker

```bash
$ git clone git@github.com:ulises-jeremias/proyecto-deep-learning-2019.git
$ cd proyecto-deep-learning-2019
$ yarn
$ yarn docker:build
$ yarn docker:start
```
